# Software-de-estoque
App criado para gerenciamento de produtos no inventário.

## Funcionalidades
O app permite a adição de um determinado produto, informando a sua quantidade, o seu valor e o nome do produto em questão adicionando esse dado a base de dados 

Foi utilizado Tkinter como a interface gráfica do projeto, para a base de dados foi utilizado postgreSQL para armazenar as informações dos produtos, a quantidade , o valor e o 
produto em questão.
Foi adicionado também uma funcionalidade que quando um determinado produto ter sua quantidade igual a 0 um email será enviado informando necessária uma nova remessa de produtos,
enviado para o e-mail informado pelo usuário.
